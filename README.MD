# auto-pet-feeder
Keep your pets happy.

Made by teralove

Updated by TerableCoder

## Info
- You must have Pet Treats or Pet Food in your inventory.
- Auto-feed triggers when pet has 5 remaining energy or less.

## Usage
### `autopetfeeder`
- Toggle on/off
- Default is on
### `feedpet`
- Feed your pet manually, same as using an item.
